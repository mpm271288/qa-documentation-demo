# 🐞 Reporte de Defectos

## DF001 – Mensaje de error genérico
- **Módulo:** Login  
- **Descripción:** Al ingresar credenciales inválidas, el mensaje de error no especifica si fue el email o la contraseña.  
- **Gravedad:** Media  
- **Estado:** Reportado  
- **Reproducible:** Sí

- **Gravedad:** Alta
